## ⚠️ 필수 규칙 (상위 문서 참조)

마케팅 활동 시 반드시 준수해야 할 규칙입니다.

### 1. UX Writing - 마케팅 문구 (🔴 필수)

**모든 마케팅 문구는 ~해요체를 사용해야 합니다.**

**적용 대상**:
- ✅ 푸시 알림 메시지
- ✅ 외부 광고 소재의 문구
- ✅ 토스 홈 광고 문구
- ✅ 프로모션 안내 문구

**올바른 예시**:
```
✅ "새로운 이벤트가 시작됐어요! 지금 바로 확인해보세요"
✅ "매일 접속하고 포인트 받아가세요"
✅ "친구를 초대하면 보상을 드려요"
```

**잘못된 예시**:
```
❌ "새로운 이벤트가 시작되었습니다"
❌ "매일 접속하고 포인트 받아가십시오"
❌ "친구 초대 시 보상 제공"
```

**참조**: [../03-design/03-ux-writing.md](../03-design/03-ux-writing.md)
