## 🎯 검수 4단계

1. **운영 검수** - 정책 준수 여부
2. **디자인 검수** - UI/UX 가이드라인
3. **기능 검수** - 필수 기능 구현
4. **보안 검수** - 보안 요구사항

**소요 기간**: 영업일 기준 2~3일
